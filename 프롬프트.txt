📈 **핵심 퀀트 공식 및 타점 락온 (V14 & V-REV & AVWAP)**
봇의 모든 매수/매도 타점 연산 시 아래 공식을 100% 팩트로 적용하라.

💎 **오리지널 무한매수법 (V14) 핵심 공식**
🔹 1회 예산 : Portion = Seed / Split
🔹 T값 (절대 진행률) : T = (Total Holdings * Average Price) / Portion
🔹 목표 익절가 : Target = Average Price * (1 + Target Ratio)
🔹 별값 (Star Price) 연산 :
🔸 감가상각 계수 : DF = 2 / Split (Split이 0이면 0.1 적용)
🔸 별% : Star Ratio = Target Ratio - (Target Ratio * DF * T)
🔸 별값 : Star Price = Average Price * (1 + Star Ratio)
🔹 매수 타점 (0주) : Buy_1, Buy_2 = MAX(0.01, PrevClose * 1.15 - 0.01) (예산 50%씩)
🔹 매수 타점 (전반전, T < Split / 2) :
🔸 평단 매수 : Buy_Avg = MIN(Average Price, Star Price) - 0.01 (예산 50%)
🔸 별값 매수 : Buy_Star = Star Price - 0.01 (예산 50%)
🔹 매수 타점 (후반전, T >= Split / 2) :
🔸 통합 별값 매수 : Buy_Star = Star Price - 0.01 (예산 100% 단일 버킷 통합)
🔹 보너스 덫 (심해 줍줍, Jubjub) :
🔸 발동 조건 : 기본 매수 주문 수량(Q_Base)이 1주 이상일 때
🔸 타점 역산 : Jub_Price = ROUNDDOWN(1회 예산 / (Q_Base + n), 2) (n은 1부터 5까지)
🔸 장전 방식 : 산출된 Jub_Price가 $0.01 초과 시, 각각 1주씩 총 5개의 지정가(LOC) 덫 강제 장전
🔹 매도 타점 :
🔸 쿼터 익절 : 보유 수량의 25% (ROUNDUP(Qty / 4)) 물량을 Star Price에 매도
🔸 목표 익절 : 잔여 수량을 Target Price에 매도

🔹 ♻️ **V14 서브 로직 리버스 모드 (소진 방어 및 복구 엔진) 핵심 공식**
🔸 발동 조건 및 자금 통제 : T > Split - 1 (20분할은 T > 19, 40분할은 T > 39) 돌파 시 즉시 전환. 남은 잔금은 리버스 모드 예산으로 100% 이관되며, 원금(Seed) 추가 투입(Zero-Injection)은 시스템 전역에서 영구 차단한다.
🔸 리버스 별값 (Rev_Star) 연산 : 직전 5거래일 종가의 산술 평균(SMA 5). (1일 차 매도 시에는 미적용).
🔸 1일 차 (소진 당일) 타격 방식 : 쿼터 매수 전면 차단. 무조건 매도(MOC) 방식으로 타격하며, 매도 수량은 전체 보유 개수의 N등분 (20분할 = 10등분 / 40분할 = 20등분, 소수점 이하 내림 강제).
🔸 2일 차 이후 타격 방식 (종료 시까지 무한 반복) : 총 잔금(이월 잔금 + 매도 대금) / 4 만큼의 예산을 Rev_Star(별지점) 아래에서 매수 시도. 직전 보유 개수의 N등분(20분할 = 10등분 / 40분할 = 20등분, 내림 적용) 수량을 Rev_Star(별지점) 위에서 매도 시도.
🔸 T값 동적 스케일링 연산 : 20분할 무매는 매도 시 T = 직전 T * 0.9, 매수 시 T = 직전 T + (20 - 직전 T) * 0.25 / 40분할 무매는 매도 시 T = 직전 T * 0.95, 매수 시 T = 직전 T + (40 - 직전 T) * 0.25.
🔸 탈출 및 일반 모드(V14) 복귀 조건 : 당일 종가가 보유 평단 대비 TQQQ는 -15% 이내, SOXL은 -20% 이내로 진입한 것을 확인하면 리버스 모드를 즉시 종료. 리버스 모드에서 연산된 최종 T값을 그대로 상속받아 일반 모드로 복귀. 복귀 후 1회 매수금(Portion)은 '현시점 총 잔금 / (Split - T)' 공식으로 타점 재조정. 일반 모드 복귀 후 잔여 예산이 1회 매수분 미만으로 다시 소진될 경우 즉각 리버스 모드로 재돌입.

⚖️ **역추세 하이브리드 (Reversion, V-REV) 핵심 공식**
🔹 1일 고정 예산 : Budget = Seed * 0.15
🔹 매수 앵커 및 타점 :
🔸 0주 새출발 : Buy_1 = PrevClose * 1.15, Buy_2 = PrevClose * 0.999
🔸 기보유 상태 (L1_Price 앵커 락온) : Anchor = L1_Price, Buy_1 = Anchor * 0.998, Buy_2 = Anchor * 0.993
🔹 당일 졸업 후 새 사이클 강제 재진입 (Same-Day Re-entry) : 프리장 개장(04:00 EST)부터 정규장 마감 45분 전(15:15 EST) 사이에 전량 익절(졸업)이 발생하여 잔고가 0주가 될 경우, 1회분 1일 고정 예산(15%)을 투입하여 당일 정규장 종가(LOC) 매수 주문을 장전하여 새로운 사이클 시작한다.
🔹 매도 타점 (LIFO 단일/다중 지층 분기 탈출) :
🔸 단일 지층 (1개 층 보유) : 상위층 덫 소각, 오직 1층 탈출망(Sell_L1 = L1_Price * 1.006) 단독 장전
🔸 다중 지층 (2개 층 이상 보유) : 1층 탈출망(Sell_L1 = L1_Price * 1.006) 및 상위층 통합 탈출망(Sell_Upper = Upper_Price * 1.010) 동시 장전
🔹 자전거래 방어막 캡핑 : MinSell = MIN(Sell_L1, Sell_Upper). 매수 타점이 MinSell 이상이면 MAX(0.01, MinSell - 0.01)로 하향 조정. (단일 지층일 경우 MinSell = Sell_L1)
🔹 평단가 리앵커링 (수익금 단가 차감 & KIS 원장 100% 디커플링) :
🔸 KIS 증권사 평단가(AVWAP 오염)를 절대 참조하지 않고 오직 순수 로컬 큐(Queue) 장부만 신뢰한다.
🔸 하위 지층 단가 상승 패러독스 원천 차단 : 1층 익절 후 잔여 지층이 2개 이상일 때는 개별 평단가를 100% 보존한다.
🔸 단일 지층 마법 격발 : 오직 잔여 지층이 단 1개 남았을 때만(len(q) == 1), 회수된 현금을 투입하여 잔여 악성 지층의 매수가를 하향 재조정한다 : New_Price = MAX(0.01, (Pure_Total_Invested - Net_Realized_Cash) / Remaining_Qty). 단, Net_Realized_Cash는 왕복 수수료 및 슬리피지 버퍼 0.6%를 선차감한 순수 회수금(Realized_Cash * 0.994)이다.
🔹 자체 VWAP 1분 슬라이싱 엔진 (가중치 기반 분할 타격) :
🔸 타임라인 : 15:27 EST ~ 15:57:59 EST (30분 타격 + 1분 클린업 가동)
🔸 가중치 분배 : 5개년 누적 거래량 프로파일(VWAP_PROFILES) 기반 매 1분 단위 타겟 수량 할당.
🔸 타격 방식 : KIS 증권사 알고리즘 위임을 전면 소각하고, 매 1분마다 산출된 할당 수량을 1호가 지정가(LIMIT)로 직접 쪼개어(Slicing) 전송.
🔸 무덤핑 및 정밀 요격 (수동 덫 소각) : 목표가를 충족시키기 위해 가격을 강제 훼손하는 막판 덤핑(슬리피지 허용) 영구 소각. 매수/매도 1호가가 목표 타점을 충족할 때만 100% 즉시 타격하며, 미충족 시 1주도 사지 않고 철저한 관망세 유지.
🔸 15:57 Clean-up 페이즈 : 15:57 도달 시 미체결 덫 취소 및 수거만 진행하고 신규 타격은 전면 바이패스(Bypass)하여 안전한 SHUTDOWN 사수.
🔸 자본 잠김 패러독스 방어 (Delay & Transfer) : 암살자(aVWAP)가 가용 현금을 점유 중일 경우, 15:27 슬라이싱을 전면 무효화(Bypass)하고 본진의 타격 플랜을 16:01 EST 애프터장으로 지연 이관하여 원자적 보존한다.
🔸 애프터장 일괄 타격 (16:01 EST) : 15:59 암살자 덤핑으로 예수금이 복구되면, 16:01 애프터장에 기상하여 이관된 본진 플랜을 100% 일괄 지정가(LIMIT)로 병합 타격한다. 유동성이 마르는 애프터장 특성을 고려해 슬라이싱은 배제한다.
🔹 Gap Hijack (자율주행 양방향 섀도우 오버라이드) 엔진 :
🔸 가동 전제 조건 (V-REV 100% 락온) : 본 하이재킹 엔진은 로컬 큐(Queue) 장부 아키텍처에 절대적으로 의존하므로, 오직 **역추세 하이브리드(V-REV)** 모드로 운용 중일 때만 가동된다. 큐 장부를 사용하지 않는 오리지널 무한매수(V14) 모드일 경우 하이재킹 엔진 진입을 시스템 전역에서 100% 영구 차단(Bypass)하라.
🔸 발동 조건 : 15:27 ~ 16:00 EST 구간, 레버리지 본종목(SOXL) 당일 1분봉 누적 VWAP 대비 현재가가 임계치(±2.0%)를 이탈하는 순간 즉각 격발.
🔸 큐 장부 절대 락온 (Zero-Sell Shield) : KIS 서버의 증권사 실잔고 조회를 전면 소각하고, 오직 **로컬 큐(Queue) 장부에 기록된 수량 합산**만을 매도 대상으로 100% 락온한다. 큐 장부상 보유 수량이 0주(새출발 상태)일 경우 어떠한 매도(SELL) 로직도 격발되지 않도록 즉시 바이패스(Bypass) 처리하여 사용자 개인 장기 물량 훼손 사고를 원천 방어한다.
🔸 상방 하이재킹 (+2.0% 상향 돌파 시 - 전량 확정 익절 스윕) :
▫️ 슬라이싱 엔진 무효화 (Hijacking) : V-REV 모드는 KIS 서버에 예약 주문을 장전하지 않으므로 기존 덫 취소 통신은 불필요하다. 즉시 로컬 슬라이스 상태 파일(hijacked=True)을 원자적 오버라이드하여 진행 중인 1분 분할 매매(Slicing)를 전면 차단하라.
▫️ 큐 장부 스윕(Sweep) 익절 : 로컬 큐 장부의 합산 수량만큼만, 실시간 매수 1호가(Bid Price)를 스캔하여 일괄 지정가(LIMIT) 매도 타격함으로써 즉시 전량 확정 익절.
▫️ 큐 상태 오버라이드 : 매도 체결 직후 로컬 큐 장부 상태를 0주로 즉시 원자적 덮어쓰기.
🔸 하방 하이재킹 (-2.0% 하향 이탈 시 - 바닥 유동성 100% 흡수) :
▫️ 슬라이싱 엔진 무효화 (Hijacking) : 즉시 로컬 슬라이스 상태 파일(hijacked=True)을 원자적 오버라이드하여 진행 중인 1분 분할 매매(Slicing) 신규 진입을 전면 차단하라. (예약 덫 취소 통신 불필요).
▫️ 영끌 스윕(Sweep) 매수 : 1일 고정 예산(15%) 중 미소진된 잔여 예산 전액을 긁어모아, 실시간 매도 1호가(Ask Price)로 일괄 지정가(LIMIT) 매수 타격.

⚔️ **순수 돌파/추종 데이 트레이딩 (암살자 aVWAP 제로-오버나잇 모듈) 핵심 공식**
🔹 가동 전제 조건 (종목 및 모드 절대 락온) : 본 암살자 모듈은 오직 SOXL 종목을 V-REV(역추세) 모드로 운용할 때만 가동된다. 타 종목(TQQQ 등) 또는 V14 모드일 경우 시스템 전역에서 100% 영구 차단하라.
🔹 예산 락온 (주문가능금액 95% 타격 및 5% 증거금 버퍼) : 암살자 예산 연산 시 가상의 '시드(Seed)' 개념 및 85% 배분 논리를 시스템 전역에서 100% 영구 소각 단행. 사용자의 실제 계좌 운용 환경을 그대로 반영하여, 실제 격발 시점의 **'KIS 실시간 주문가능금액(Cash)'의 95%를 최종 타격 예산으로 강제 락온**한다. 이를 통해 수수료 차감 및 증거금 연산 딜레이로 인한 API 리젝(Reject)을 완벽히 차단하고, 5%의 안전 버퍼를 제외한 가용 현금 95%를 하루 1회에 한하여 투입한다(1-Shot 1-Kill).
🔹 세션별 독립 VWAP 동적 연산 : 전일 종가의 영향을 100% 배제하고, 당일의 순수 거래대금/거래량 기반 VWAP만 추적한다.

🔸 통합 및 강제 렌더링 (일괄 타결) : 위 3단계 타격에서 발견된 모든 취약점(특히 들여쓰기 및 문법 오류)을 내부 메모리에서 즉시 보완(수정 승인 간주)하라. 그 후 '무엇이 문제였고 어떻게 방어 코드를 주입했는지' 5줄 이내의 이모지 핵심 요약 브리핑을 최상단에 타전하고, 직후 합의된 100% 풀 코드를 단일 코드 블록으로 생략 없이 렌더링하라. (중간 보고 및 승인 대기 절대 금지)
